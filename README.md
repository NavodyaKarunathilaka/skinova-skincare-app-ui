# 🌸 Skinova - Skincare Companion App UI Design

Skinova is a comprehensive skincare application designed to help users discover personalized skincare routines, track their skin health, and purchase recommended products. This repository contains the Figma UI design showcasing the main navigation structure, user flows, and interface components.

## ✨ Overview

Skinova provides:

- **Personalized skincare routines** based on user skin types and concerns
- **Product discovery and e-commerce features**
- **User account management** (Login/Signup)
- **Routine tracking** (morning/night)

The design focuses on a **clean, minimalist, and user-friendly interface** to ensure intuitive navigation for all skin types and skincare goals.

---

## 🎨 **Key UI Components**

### 🧭 **Navigation Bar**
- Logo (top-left)
- Main navigation options:
  - Home
  - Skin Profile
  - Skin Types (Dry, Oily, Normal, Combination)
  - Problem Areas
  - Routine sections (Routin1, Routin2)
  - Shopping (Product listings, Shopping Cart, Payment)
  - User Account (Login, Signup)
- Utility Icons:
  - Search
  - User Profile
  - Communication/Notification

### 🗂 **Skin Type Focus**
- Dedicated sections for each skin type:
  - Dry
  - Oily
  - Normal
  - Combination
- Problem area identification

---

## 📝 **User Flow**
1. User identifies their **skin type**.
2. Selects **problem areas**.
3. Receives **customized routines** (Routin1 and Routin2).
4. Browses recommended **products**.
5. Completes purchase through the integrated **payment system**.



---

## 📌 **Tech & Design Stack**
- **Design Tool:** Figma
- **Design System:** Custom minimalist skincare UI kit
